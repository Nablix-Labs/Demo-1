# Rules

Hard constraints for this project. Not suggestions.

## Commits
- **Never** add a `Co-Authored-By: Claude` trailer (or any Claude/AI attribution) to commits or PRs.

## UI / design
- **Monochrome palette.** Greyscale UI per the design tokens in `globals.css`. The only sanctioned colour outside greyscale is the tutor's red ink (`#b42318`) and the student's own drawing-tool colours on the canvas.
- **No emojis.** No emoji glyphs in UI copy, labels, or text.
- **Tasteful icons OK.** The existing lucide-react icon set (nav rail, toolbar, status) is part of the design and stays. Don't add decorative/novelty icons that don't earn their place.
- **No decorative gradients.** No gradient fills for decoration. Functional gradients are allowed where they aren't decorative — e.g. the 1px grid-line backgrounds on canvases.
